# 🛡️ Shell we dance | SK Shieldus Rookies

> **"Shell we dance?"** - 보안의 리듬에 맞춰 취약점을 찾아내고 안전한 금융 인프라를 구축하는 8인의 화이트해커 팀입니다.

---

### 🏦 Project: 금융 서비스 진단 및 모의해킹
본 프로젝트는 실제 은행 업무가 가능한 Web/App 환경을 직접 개발하고, 금융권에서 발생할 수 있는 치명적인 취약점을 **공격자 관점(Red Team)**에서 분석하여 **방어 전략(Blue Team)**을 수립하는 것을 목표로 합니다.
- **Target**: 자체 개발한 뱅킹 서비스 (Web & Android App)
- **Focus**:
  - Secure Coding: 개발 단계에서의 보안 가이드 준수 여부 확인
  - Vulnerability Diagnosis: OWASP Top 10 및 금융권 특화 취약점 진단
  - Scenario Hacking: 비정상 이체, 개인정보 탈취 등 실무형 공격 시나리오 수행
